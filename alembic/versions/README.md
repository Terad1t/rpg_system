This folder will contain Alembic migration scripts generated by `alembic revision --autogenerate`.
Run the following to create the initial migration after configuring `alembic.ini`:

1. Set the SQLAlchemy URL (either in `alembic.ini` as `sqlalchemy.url` or set `DATABASE_URL` env var):

   export DATABASE_URL="sqlite:///./rpg_system.db"

2. Generate initial revision:

   alembic revision --autogenerate -m "initial"

3. Apply migrations:

   alembic upgrade head
